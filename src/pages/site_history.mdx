My site's been through some stuff, here's a somewhat comprehensive history and some director's commentary. If you'd like to check out the full history you can view it on its [Github page](https://github.com/moosyu/moosyu.github.io).

## Initial version (December 2022)

The first version of my site was made right after seeing [Daniel's Virtual Place](https://displayman.neocities.org/) for the first time so it really tried to follow his kind of design vibes the best I could even though in the end the similarities were pretty superficial. Even still in the end I'm still pretty proud of the result considering prior to its creation I didn't know a lick of HTML or CSS. This isn't to say the code was done well at all. I was basically freestyling it and you could tell, for some reason I thought `<body>` was the same as `<main>` so left `<header>` and `<footer>` outside of the body and I bizarrely chose to separate attribute equal signs with a space on both sides for some reason.

![2022 site](/assets/site_history/dec-2022.png)

## Final pre-ssg version (August 2024)

Cut two years, by this point I'd been working on the site for almost two years and consistently updating it for about one so some pretty big changes had been made. Most importantly I actually figured out what was going on HTML-wise, my JS was still coming along though as most of it was either stolen or garbage, some things never change I guess. This site was even pretty responsive for mobile which was a little bit of a surprise going back to it.

![2024 site index](/assets/site_history/aug-2024.png)

This was the index page from that period, as you can see we really snazzed stuff up. There's a lot going on but that's the look I was going for at the time and I think I did a really good job with it to be honest. As I'm sure you can see the comments could use some works though and it's an issue that I'm still struggling to deal with to this day. There's a proper navbar now with stuff and it even has drop-downs which is something I still kind of dislike having to make work so I'm pretty surprised me from back then pulled it off.

![2024 site reviews](/assets/site_history/aug-2024-reviews.png)

This is my old reviews (thoughts) page from before I realized I'm nowhere near good enough at writing anything to be calling them reviews. It's styled after MAL and I'm just now realizing it has the same background colour as my current content background colour. I had been wondering where I got that colour from, pretty cool. This was also before I decided that it would probably be a good idea to have a consistent theme so it was kind of just random bullshit on some pages. Most of them it was just the same blue with some memes, which while fun got a bit grating for me personally and as the largest majority of views on my site I think I get to decide when to change it up.

![2024 site comic reviews](/assets/site_history/aug-2024-reviews-comics.png)

This is the comics reviews page of that period. I kept them split up during that time because I had yet to figure out pagination and the hundreds of entries had to be loaded by the client so it was the only way to avoid blowing up people's PCs and it still barely worked. This was also when I was using decimals that weren't just 0.5 which was dumb as hell as its pretty tough to differentiate between a 7.3 and a 7.5 so it just becomes pointless. Despite all my complaining I did like this version of this page when I could just write random bullshit and call it a day. I know nobody is forcing me to actually think about what I'm writing these days but I still feel like I should at least give a shot at saying what I think about something someone has put time into. I like the Sandy Loam pictures too.
