I've seen some people want a little script that can emulate a VN's dialog which is what this does. Check out the example if you'd like to see exactly what I mean. This is obviously missing a lot of stuff obviously as it only allows moving through dialog, changing profiles and dialog paths. I don't intend to add anything else apart from maybe changing backgrounds so if you need anything other than that I highly recommend checking out [Ren'Py](https://www.renpy.org/) which is a visual novel game engine that can export to HTML5.
